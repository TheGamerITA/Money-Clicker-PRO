# Money Clicker PRO

Versione PRO del clicker con:

- Canvas 2D
- Tab separate
- Eventi casuali
- Missioni giornaliere
- Prestige / Rebirth
- Suoni WebAudio
- Icone SVG
- Particelle e animazioni

## Come avviarlo

1. Apri la cartella in VS Code
2. Installa Live Server
3. Tasto destro su `index.html`
4. Open with Live Server

## File principali

### `js/data.js`
Qui modifichi:
- upgrade
- obiettivi
- configurazione
- tipi di missione

### `js/icons.js`
Qui puoi aggiungere o modificare icone SVG.

### `js/audio.js`
Qui puoi modificare i suoni.

### `js/script.js`
Qui c'è la logica principale:
- canvas
- eventi
- missioni
- prestigio
- UI
- salvataggio
