# Money Clicker

Versione del clicker con:

- Canvas 2D professionale
- Sprite procedurali
- Sfondi animati a livelli
- Boss click
- Upgrade avanzati
- Albero delle abilità
- Mini-giochi
- Roulette
- Slot machine
- Eventi a scelta multipla
- Missioni giornaliere
- Missioni settimanali
- Achievement progressivi
- Ricompense speciali
- Sidebar moderna
- Notifiche animate
- Modali
- Temi sbloccabili
- Prestige / Rebirth

## Come avviarlo

1. Apri la cartella in VS Code
2. Installa Live Server
3. Tasto destro su `index.html`
4. Open with Live Server
